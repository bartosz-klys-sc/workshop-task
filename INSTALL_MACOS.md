# macOS Install (Unsigned)

You will receive two files:
- `PDF Generator-0.0.0.dmg` (Intel Macs)
- `PDF Generator-0.0.0-arm64.dmg` (Apple Silicon / M1–M3 Macs)

## Which file should I use?
- If you have an M1/M2/M3 Mac: use the **arm64** file.
- If you have an Intel Mac: use the **non-arm64** file.

If you are unsure, click  → About This Mac and check the chip:
- **Apple M1/M2/M3** = arm64
- **Intel** = non-arm64

## Install steps
1. Double‑click the `.dmg` file to open it.
2. Drag **PDF Generator** into **Applications**.
3. Open **Applications** and find **PDF Generator**.
4. **Right‑click → Open** (first launch only).
5. Confirm the security prompt (unsigned app).

## Troubleshooting
- If macOS blocks the app, go to **System Settings → Privacy & Security** and allow it.
- If it won’t open after moving to Applications, remove it and repeat the steps above.
