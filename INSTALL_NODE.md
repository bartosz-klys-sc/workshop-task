# Local Run (Node.js)

This option runs the tool locally in a browser with a one‑click launcher.

## Prerequisites
- Node.js LTS must be installed (ask IT if needed).

## Run
1. Double‑click `start.command`.
2. Your browser will open to `http://localhost:5173/`.
3. Keep the terminal window open while using the tool.
4. Press Enter in the terminal window to stop the server.

## Troubleshooting
- If you see a security warning, right‑click `start.command` → Open.
- If the browser doesn’t open, paste `http://localhost:5173/` into the address bar.
